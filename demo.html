Lightbox

Simply add a rel="lightbox" to any link that has a href with an image link. Any alt attached a nested image/thumbnail will show as a caption in the lightbox. By default, an automatic gallery will be created with next/previous controls will be added for all found rel="lightbox" anchors. Example:

Gallery

Alternatively you can load multiple images without having defined anchors/thumbnails. Simply add a rel="gallery", assign a unique id, and include data-gallery in the form of "url,caption;url,caption". Example:

Gallery Link
